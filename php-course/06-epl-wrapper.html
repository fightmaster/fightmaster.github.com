Обертка над epl

Описание

opensoft/epl - библиотека-обертка над языком EPL. Используется для работы генерации кода для принтеров на ProFIT.

Аналоги

На момент написания библиотеки достойных аналогов не было.

Целевая аудитория

Все, кому необходимо взаимодействовать с термо-принтерами.

Задачи

Библиотека написана достаточно давно. наверняка в ней есть, чего улучшить. Для начала, можно упростить валидацию, применив assert.
