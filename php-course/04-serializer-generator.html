Генератор роутинга

Описание

Если взять serializer, то ему нужны настройки.

MyBundle\Resources\config\serializer\ClassName.yml
    Fully\Qualified\ClassName:
        properties:
            some-property:
                expose: true
                type: string
                serialized_name: foo
                since_version: 1.0
                until_version: 2.0
                groups: ['get','patch']

Это актуально как и для opensoft/simple-serializer, так и schmittjoh/serializer.
